## 🌟 核心特性 (Key Features)

### 🧬 原生系统控制 (Native System Control) **[REFACTORED]**
告别“模拟”与“空谈”，AI 拥有真正的系统操作权限。
*   **真实路径解析**：AI 能够理解 `桌面`、`文档`、`下载` 等自然语言路径，并自动映射到您的真实系统目录（如 `C:\Users\You\Desktop`）。
*   **文件操作**：直接读写文件、创建文件夹、删除无用数据。
*   **CMD/PowerShell**：内置命令执行能力，可运行任何 Windows 命令或脚本。


### 🧬 自我进化 (Self-Evolution) **[NEW]**
AI 不再是被动的工具，它具备了自我思考与成长的能力。
*   **动态技能生成**：AI 可以编写 Python 代码并将其保存为新技能（Skill），即时扩展自身能力。
*   **夜间自动进化**：当您休息时（系统空闲 2 小时），AI 会自动唤醒，尝试实现日记中的计划（如编写新代码、学习新技能），并在完成后向您汇报成果。
*   **日记与反思**：每次对话结束后，AI 会自动反思自己的不足，并将改进计划写入日记。

### 🧠 1052 协议 (Memory & Experience)
内置 **[1052 协议](1052_Protocol/README.md)**，赋予 AI 长期记忆与经验积累能力。
*   **记住偏好**：AI 会记住您的称呼、语言习惯和工作偏好。
*   **经验学习**：AI 会自动总结解决问题的经验，并在下次遇到类似问题时自动调取。
*   **跨会话记忆**：即使重启程序，AI 依然记得您。

### 🔌 多端接入 (Multi-Platform) **[NEW]**
*   **Web UI**：现代化卡片式设置界面，响应式设计，支持夜间模式。
*   **Feishu (飞书) 机器人**：配置 App ID 和 Secret，即可在飞书群或私聊中与 AI 互动。
*   **Telegram 机器人**：支持 Telegram Bot API，随时随地远程控制您的电脑。
    *   **指令支持**：发送 `/new` 即可重置当前会话记忆。

### ⏰ 智能调度与提醒
*   **精准时间感知**：修复了时区问题，AI 现在能准确获取系统本地时间（UTC+8 等）。
*   **自然语言定时**：只需说“10分钟后提醒我喝水”或“明早8点叫我”，AI 会自动设置系统级定时任务。

## 🚀 快速开始

### 方式一：源码运行 (推荐开发者)

1.  **克隆项目**：
    ```bash
    git clone https://github.com/your-repo/1052.git
    cd 1052
    ```
2.  **安装依赖**：
    ```bash
    pip install -r requirements.txt
    ```
3.  **运行**：
    ```bash
    python app.py
    ```
4.  **访问**：浏览器打开 `http://127.0.0.1:10052`。

### 方式二：Exe 版本 (推荐普通用户)

1.  **下载**：获取最新版本的 `1052AI.exe`。
2.  **部署**：将其放入一个**空文件夹**（例如 `MyBot`，因为程序会生成数据文件）。
3.  **运行**：双击 `1052AI.exe`。
4.  **配置**：
    *   点击左下角 **设置 (Settings)**。
    *   输入您的 **API Key**（默认支持 SiliconFlow / DeepSeek）。
    *   **系统控制权限**：开启以允许 AI 操作电脑。
    *   **社交平台接入**：在设置页面配置飞书或 Telegram 机器人参数。

## 🛠️ 功能指南

### ⚙️ 系统设置
*   **模型配置**：支持 OpenAI 格式接口，默认配置 **SiliconFlow (硅基流动)** 的 DeepSeek-V3 模型。
*   **社交接入**：
    *   **Feishu**：填入 App ID 和 App Secret，配置事件订阅请求地址为 `http://your-ip:10052/api/feishu/event`。
    *   **Telegram**：填入 Bot Token，即可启动 TG 机器人。

### 🧠 记忆与进化
*   **记忆**：直接告诉 AI “我喜欢 Python”，它会调用 `protocol_remember` 记住。
*   **进化**：当 AI 遇到无法解决的问题时，它会记录在案。在夜间空闲时，它会尝试编写脚本来解决这个问题。

### 📁 技能扩展 (Skills)
*   **原生技能**：AI 已内置文件读写、CMD 执行等核心能力。
*   **自定义技能**：将 Python 脚本放入 `skills/` 文件夹，AI 会自动加载。
*   **AI 生成技能**：告诉 AI “帮我写一个爬虫技能”，它会自动生成代码并保存到 `skills/`。

## 📂 项目结构

*   `app.py`: 核心服务端源码。
*   `core_skills.py`: **核心技能库**（原生系统控制、文件操作、路径解析）。
*   `telegram_utils.py`: Telegram 机器人实现。
*   `feishu_utils.py` (集成在 app.py): 飞书机器人实现。
*   `skills/`: **技能目录**（用户或 AI 生成的扩展脚本）。
*   `1052_data/`: 存储记忆、经验和用户数据。
*   `chat.db`: 聊天记录与系统日志数据库。
*   `templates/` & `static/`: 前端界面资源。
