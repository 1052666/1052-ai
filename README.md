# 1052 AI - 本地优先的 AI 智能体框架

> **The Chinese Version of OpenClaw | 深度适配 Windows 的 AI 智能体**  
> 具备 **自我进化**、**长期记忆** 与 **系统控制** 能力的下一代桌面助手。

1052 AI 是一个基于 Flask 构建的强大本地 AI 智能体框架。它不仅是一个聊天机器人，更是一个能够操作计算机、管理记忆、编写代码并自我进化的智能体。

## 🌟 核心特性 (Key Features)

### 🧬 自我进化 (Self-Evolution) **[NEW]**
AI 不再是被动的工具，它具备了自我思考与成长的能力。
*   **日记与反思**：每次对话结束后，AI 会自动反思自己的不足，并将改进计划写入日记。
*   **夜间自动进化**：当您休息时（系统空闲 2 小时），AI 会自动唤醒，尝试实现日记中的计划（如编写新代码、学习新技能），并在完成后向您汇报成果。
*   **代码编写**：AI 拥有直接编写和运行 Python 代码的能力，可以为自己创造新的工具。

### 🧠 1052 协议 (Memory & Experience)
内置 **[1052 协议](1052_Protocol/README.md)**，赋予 AI 长期记忆与经验积累能力。
*   **记住偏好**：AI 会记住您的称呼、语言习惯和工作偏好。
*   **经验学习**：AI 会自动总结解决问题的经验，并在下次遇到类似问题时自动调取。
*   **跨会话记忆**：即使重启程序，AI 依然记得您。

### 💻 深度适配 Windows
专为 Windows 用户打造的生产力工具。
*   **CMD/PowerShell 控制**：内置 CMD 控制技能，可执行系统命令、打开应用。
*   **文件系统操作**：支持直接读写本地文件，支持环境变量（如 `%USERPROFILE%`）。
*   **任意代码执行**：支持运行任意 Python 代码，进行复杂计算或数据处理。
*   **一键运行**：提供 `.exe` 版本，无需 Python 环境，开箱即用。

### ⏰ 智能调度与提醒
*   **自然语言定时**：只需说“10分钟后提醒我喝水”或“明早8点叫我”，AI 会自动设置系统级定时任务。
*   **精准时间感知**：AI 实时感知当前时间，准确处理相对时间请求。

### 🔌 全面集成 MCP (Model Context Protocol)
支持标准化的 MCP 协议，轻松连接外部世界。
*   **兼容性**：支持任何符合 MCP 标准的 Server（如 GitHub、Google Drive 等）。
*   **本地扩展**：只需将 Python 脚本放入 `skills` 文件夹，即可为 AI 添加新能力。

## 🚀 快速开始 (Exe 版本) - 推荐

1.  **下载**：获取最新版本的 `1052AI.exe`。
2.  **部署**：将其放入一个**空文件夹**（例如 `MyBot`，因为程序会生成数据文件）。
3.  **运行**：双击 `1052AI.exe`。
    *   程序会自动初始化环境，生成 `skills`、`1052_data` 等目录。
    *   自动打开默认浏览器访问 `http://127.0.0.1:10052`。
4.  **配置**：
    *   点击左下角 **设置 (Settings)**。
    *   输入您的 **API Key**（默认支持 SiliconFlow / DeepSeek）。
    *   开启 **系统控制权限**（System Control）以允许 AI 操作电脑。
    *   开启 **自我思考与进化**（Self-Reflection）以激活自动进化功能。

## 🛠️ 功能指南

### ⚙️ 系统设置
*   **模型配置**：支持 OpenAI 格式接口，默认配置 **SiliconFlow (硅基流动)** 的 DeepSeek-V3 模型。支持一键切换 Ollama 本地模型。
*   **系统控制权限**：**安全开关**。开启后 AI 才能执行 CMD 命令、修改文件和运行代码。建议在受信任环境下开启以获得完整体验。
*   **自我思考与进化**：开启后，AI 会在后台进行反思和自我改进。

### 🧠 记忆与进化
*   **记忆**：直接告诉 AI “我喜欢 Python”，它会调用 `protocol_remember` 记住。
*   **进化**：当 AI 遇到无法解决的问题时，它会记录在案。在夜间空闲时，它会尝试编写脚本来解决这个问题。第二天您会收到一份“夜间进化报告”。

### ⏰ 定时提醒
*   **指令示例**：“10秒后提醒我测试”、“明天下午3点提醒我开会”。
*   **机制**：AI 会解析自然语言，通过后台调度器精准触发提醒。

## 📂 项目结构

*   `1052AI.exe`: 主程序。
*   `app.py`: 核心服务端源码。
*   `protocol1052/`: **1052 协议** 核心实现 (记忆与经验)。
*   `skills/`: **技能目录**。
    *   `cmd_control/`: 系统命令与代码执行技能。
    *   `scheduler/`: 定时任务技能。
*   `1052_data/`: 存储记忆、经验和用户数据。
*   `chat.db`: 聊天记录与系统日志数据库。
*   `templates/` & `static/`: 前端界面资源。
